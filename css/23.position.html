Position의 이해

static

기본 상태는 static이며, 작성 순서대로 배치됨
한 번 정해진 위치는 변경 불가능함

relative

position을 relative로 변경 시 위치 조정 가능(배치 형태는 변경 불가능)
top, left, right, bottom과 z-index 속성으로 제어

absolute

position을 absolute로 설정 시 좌표를 이용하여 위치 지정 가능함
기존에 존재하는 화면 위를 덮으며 배치됨(overlay)
top, left, right, bottom 속성으로 위치 설정 가능

fixed

position을 fixed로 설정 시 Viewport를 기준으로 위치 설정 가능
따라서 문서가 이동해도 fixed로 설정된 위치는 변하지 않음 (사용자를 따라오는 요소 구현 가능)
